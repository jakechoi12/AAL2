```python
"USD": "0000001"  # 정상 작동
"JPY": "0000002"  # 정상 작동
"EUR": "0000003"  # 정상 작동
"CNY": "0000004"  # 작동하지 않음 (데이터 없음)
"GBP": "0000005"  # 작동하지 않음 (데이터 없음)
```

- **CAD (캐나다달러)**: item_code 0000013 사용 가능
- **SGD (싱가포르달러)**: item_code 0000024 사용 가능
- **NZD (뉴질랜드달러)**: item_code 0000026 사용 가능

## 참고사항

1. **USD 관련**: 여러 item_code가 USD로 식별되었지만, 실제로는 다른 달러 통화들입니다:
   - 0000013: 캐나다달러 (CAD)
   - 0000015: 홍콩달러 (HKD)
   - 0000017: 호주달러 (AUD)
   - 0000024: 싱가포르달러 (SGD)
   - 0000026: 뉴질랜드달러 (NZD)
   - 0000031: 대만달러 (TWD)
   - 0000036: 브루나이달러 (BND)

2. **GBP 관련**: 0000012는 영국파운드, 0000052는 이집트파운드입니다.

3. **데이터 없음**: item_code 0000004, 0000005, 0000006~0000011, 0000027, 0000030, 0000054~0000100은 데이터가 없습니다.
